# World-Population-EDA 🌍

Exploratory Data Analysis on World Population dataset using Python & Pandas.

## 📊 Analysis Performed
- Data overview (`info`, `describe`, `isnull`)
- Top countries by 2022 population
- Correlation heatmap
- Population trends by Continent (1970–2022)
- Boxplot distribution analysis

## 🛠️ Libraries Used
- Pandas
- Seaborn
- Matplotlib

## 📁 Dataset
Source:[Kaggle - World Population Dataset](https://www.kaggle.com/code/hasibalmuzdadid/world-population-analysis/input))

## 🚀 How to Run
1. Clone the repo
2. Install libraries: `pip install pandas seaborn matplotlib`
3. Open `Exploratory Data Analysis in Pandas.ipynb`
